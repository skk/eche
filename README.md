# eche [![Build Status](https://travis-ci.org/skk/eche.svg?branch=master)](https://travis-ci.org/skk/eche)

## Description

A lisp-like Language interpreter.  It is based on the
[make-a-lisp](https://github.com/kanaka/mal) guilde.  eche has completed steps 0 through 2 of [make-a-lisp](https://github
.com/kanaka/mal/blob/master/process/guide.md).
